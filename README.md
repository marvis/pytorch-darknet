# pytorch-darknet
This repository supports training classification task on pytorch by loading darknet defined cfg and weights.

# Resnet50 evaluation
- pytorch: * Prec@1 76.012 Prec@5 92.934
- darknet: * Prec@1 75.798 Prec@5 92.804
